direction for the boss and class redesigns — earmarked for SPLASH / COVER
/ loading art. These are the full concept ILLUSTRATIONS; the in-game
pixel sprites are derived from them separately (PixelLab), so this is the
high-res source to draw splash art from.

companion to `boss_saint_varo`.

## Before shipping as splash art — checklist
1. **Steam AI-content disclosure** is required for AI-generated assets.
2. **Fake signatures:** FLUX sometimes hallucinates a fake artist signature
   in a corner (seen on some drafts). Inspect each and paint it out before
   use.
3. **Upscale:** 1024px is fine for a card/thumbnail; a full-screen splash
   may want an upscale pass (these are drafts, not final renders).
