========================================
  CROWNLESS - co-op build for friends
========================================

Hi! Thanks for playing. This is a private build - not on any store yet -
so setup is a little more hands-on than usual. It only takes a minute.


STEP 1 - EXTRACT THE WHOLE ZIP FIRST
------------------------------------
Right-click the .zip you downloaded and choose "Extract All..." Pick a
folder (your Desktop is fine) and open that folder. Do NOT run the game
from inside the zip - extract it first, then run it from the extracted
folder. Saves are kept safely in your Windows user folder, so you can
delete or replace this folder later without losing progress.


STEP 2 - RUN IT (Windows will warn once - that's expected)
----------------------------------------------------------
Double-click  Crownless.exe

The first time, Windows may show a blue box:
    "Windows protected your PC"
That is ONLY because this build isn't code-signed (signing costs money
and we're a tiny indie project, not because anything is wrong). To run it:

    Click  "More info"  ->  then  "Run anyway"

You only have to do this once.


STEP 3 - PLAY TOGETHER
----------------------
One person HOSTS, everyone else JOINS.

  HOST:  Title screen -> "Play Together" -> Host. Pick your character and
         chapter. A short JOIN CODE appears - send it to your friends
         (there's a copy button).

  JOIN:  Title screen -> "Play Together" -> Join. Type the code the host
         gave you, pick your own character, and wait in the lobby.

  When everyone's in, the host presses Start.

No router setup, no port forwarding, no "what's my IP" - ever. The join
code handles all of that behind the scenes, even on home wifi.

IMPORTANT: everyone must be on the SAME version. The version is printed
on the title screen (e.g. "build 0.1.0"). If they don't match, the game
refuses the join with a clear message naming both versions - just make
sure you all downloaded the same zip.


IF SOMETHING GOES WRONG (three quick fixes)
-------------------------------------------
* Antivirus deleted or blocked the game?  This is a known false alarm
  with games made in the Godot engine (lots of indie games hit it). The
  file is safe - restore it from quarantine and add the folder to your
  antivirus exclusions, then run again.

* Can't connect / "wrong build"?  Double-check the join code (it's easy
  to mistype), and make sure both of you are on the exact same zip.

* Laggy?  Whoever HOSTS should be the person with the best/most stable
  internet connection. Try again with a different host.


Have fun out there. - The Crownless crew
