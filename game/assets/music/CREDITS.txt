Music override folder
=====================
Drop a .ogg, .mp3 or .wav file here named after a track to replace it,
looped automatically. Track names: village, darkwood, marsh, keep,
magma, icefield, graveyard, desert, rainstorm, holy, crystalline,
title, roster, boss_fangmaw, boss_morwen, boss_vargoth.
Multi-boss brawls (2-5 live bosses) look for: boss_x2, boss_x3,
boss_x4, boss_x5 — optional; the first boss's own theme plays when a
tier's file is missing.
Delete a file to fall back to the built-in composed version.
