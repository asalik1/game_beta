Two fonts in this folder. Both SIL OFL 1.1, both free for commercial
use, and both ship their license text (required by the OFL).

PIXELIFY SANS (PixelifySans.ttf) — the UI's display/header face
  (panel titles, section headers, HUD title cards, boss bar name,
  speaker names). Body text stays the engine's default sans for
  readability.

  Copyright 2021 The Pixelify Sans Project Authors
  (https://github.com/eifetx/Pixelify-Sans)
  License: SIL Open Font License 1.1 — see OFL.txt in this folder.
  Downloaded from the Google Fonts repository
  (github.com/google/fonts, ofl/pixelifysans).

CINZEL DECORATIVE (CinzelDecorative-Bold.ttf) — the LOGO face, and
  ONLY the logo: the CROWNLESS wordmark on the boot cover (2026-07-17).
  Drawn from Roman inscriptional capitals, so it reads regal under the
  Ember Crown. It is deliberately NOT the UI face — a smooth ornate
  serif would fight the pixel chrome on every other screen.

  Copyright (c) 2012 Natanael Gama (info@ndiscovered.com),
  with Reserved Font Name 'Cinzel'
  License: SIL Open Font License 1.1 — see OFL-CinzelDecorative.txt.
  Downloaded from the Google Fonts repository
  (github.com/google/fonts, ofl/cinzeldecorative).

  RESERVED FONT NAME: the OFL forbids shipping a MODIFIED version under
  the name "Cinzel". We ship it unmodified, so this is a non-issue —
  but if anyone ever edits the TTF, it must be renamed first.

How the fonts are wired:
  scripts/ui/theme.gd — UITheme.display_font() loads Pixelify and applies
  it to titles/headers via UITheme.title()/header(); UITheme.logo_font()
  loads Cinzel and is applied ONLY by UITheme.logo(), which only
  ui/cover.gd calls. Deleting either TTF falls back to the default sans —
  nothing breaks.
