How the override system works:
  Any PNG named <sprite>.png in this folder replaces the procedural
  pixel-art for that sprite (see Art.tex in scripts/art.gd). Sprites
  are auto-scaled to a consistent on-screen size regardless of source
  resolution, so you can drop in art of any size.

  To restyle a character, replace its PNG here (warrior/archer/mage/
  assassin/paladin/warlock/wolf/direwolf/spider/cultist/skeleton/witch/king). Delete a
  PNG to fall back to the built-in procedural art.

  ANIMATION (Track C seam): drop <sprite>_anim.png — a HORIZONTAL strip
  of square frames (frame count auto-detected from width/height) — and
  that creature animates in-game (idle pace, double-time while moving).
  All *_anim.png strips here are derived from the CC0 Crawl sprites
  (2-frame breath cycles), generated by us — same public-domain status.

PIXEL CRAWLER (Anokolisa) — purchased bundle, 2026-07-06: some sprites
  (skeleton, zombie, cultist, sentry, and future roster swaps) are from
  the Pixel Crawler packs, licensed for commercial game use. Credit
  optional; the raw packs must NOT be redistributed as assets.
  Install pipeline: idle sheets tight-cropped to content and saved as
  <name>_anim.png + first frame as <name>.png (codex/portraits).
  Re-derivation (2026-07-08): 25 mobs (skeleton*, orc*, zombie/zombie_brute,
  banshee, mummy/mummy_mage, stone_golem, medusa, bandit_*, royal_soldier/
  priest, scholar_censor, elf_ranger, fungus_*) were re-cut from the pack
  side-facing Mob sheets (Idle=4f -> _anim, Run=6f -> _walk, frame0 -> base).
  IMPORTANT: frames must be SQUARE (Art._strip_info counts frames as
  width/height) — so each frame is PILLARBOXED to a square (transparent side
  pad, character bottom-anchored = feet at square base), NOT force-resized to
  a square (which squished portraits fat/squat). union-cropped across idle+run
  for a stable anchor; gamma 0.78 pre-brighten for the HDR tonemap; sheets
  already face LEFT (convention) so no mirror. Source packs: Free Pack (orcs/
  skeletons), Cemetery (zombies/banshee), Desert (mummies), Forge (stone),
  Garden (medusa), Hideout (bandits), Castle (royals), Library (scholars),
  Fairy Forest (elves), Cave (fungus).
